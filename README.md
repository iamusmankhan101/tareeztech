# Tareez Tech - Digital Marketing Agency & Web Development Company

A modern, SEO-optimized website for Tareez Tech - a leading digital marketing agency and web development company in Lahore, Pakistan.

## 🚀 Features

- **SEO Optimized**: Integrated with 30+ low-competition keywords for maximum search visibility
- **Smooth Animations**: Built with Framer Motion for engaging user experience
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Fast Performance**: Built with Vite for lightning-fast load times
- **Smooth Scrolling**: Lenis integration for buttery smooth scroll experience

## 🎯 Target Keywords

This website is optimized for the following low-competition keywords:
- Digital marketing agency in Lahore
- Best web development company in Lahore
- Custom web development services
- Digital marketing expert in Pakistan
- Web and app development services
- And 25+ more strategic keywords

See [SEO-KEYWORDS-IMPLEMENTATION.md](./SEO-KEYWORDS-IMPLEMENTATION.md) for complete keyword strategy.

## 🛠️ Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lenis** - Smooth scrolling
- **React Router** - Navigation
- **Lucide React** - Icons

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Customization

### Colors
The primary brand colors are defined in `src/index.css`:
- Primary: `#0d10d3` (Blue)
- Accent: `#00f2ff` (Cyan)
- Background: `#000000` (Black)

### Content
Update content in respective component files:
- Hero text: `src/components/Hero.jsx`
- Services: `src/components/Services.jsx`
- About: `src/components/AboutSummary.jsx`

### SEO
Update meta tags in `index.html` for:
- Title
- Description
- Keywords
